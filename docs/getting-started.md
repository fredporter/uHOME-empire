# uHOME-empire Getting Started

1. Read `docs/boundary.md`.
2. Review `docs/activation.md`.
3. Read `docs/contact-and-crm-model.md` before defining sync ownership.
4. Review `docs/webhook-automation-runbook.md` for the webhook and API lane.
5. Review `docs/provider-lane-status.md` before treating a provider as live.
6. Use `docs/provider-template-patterns.md` when adding payload mappings.
7. Inspect `src/webhooks/` for the base webhook and API contract templates.
8. Start from `src/webhooks/connection-template.json` for new integrations.
9. Put runnable demonstrations in `examples/`.
10. Run `scripts/run-empire-checks.sh`.
11. Add regression tests for any workflow contract.
