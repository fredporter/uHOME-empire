# v2.0.1 Sync Alignment

`uHOME-empire` is the always-on sync and webhook layer on top of the `v2.0.1`
platform spine.

## Contract Edges

- `uDOS-core` owns canonical workflow and capability semantics
- `uDOS-wizard` owns provider and online transport routing
- `uHOME-server` owns the base Linux host runtime and local-network services
- `omd-mac-osx-app` remains the Apple-native upstream for user and binder
  contact records

## Starter Surfaces

- `src/workflow-pattern.json` for reviewed console workflow structure
- `src/sync-contract.json` for sync channels and upstream ownership
- `src/webhooks/` for configurable webhook and provider templates

## Scope

This round defines the sync contract and webhook lane. It does not yet make the
full contact intelligence or publishing pipeline end-to-end.
